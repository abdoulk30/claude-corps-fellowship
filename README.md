# Claude Corps Fellowship Application

An interactive site built in place of a static cover letter for my [Claude Corps Fellowship](https://www.anthropic.com/) application (Anthropic × CodePath).

**Live site:** https://abdoulk30.github.io/claude-corps-fellowship/

## What's here

- Background and why I'm applying
- How I map to what the fellowship is looking for
- Projects I've shipped (VentureFlow, Librio, JobSync)
- Certificates for the two required Anthropic courses (AI Fluency: Framework & Foundations, Claude 101)
- A delegation plan for how I split this build with Claude
- A diligence statement on how AI was actually used to build this site

## Stack

Plain HTML, CSS, and JavaScript. No framework, no build step, deployed via GitHub Pages.
